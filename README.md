# Planificador — Task & Schedule Planner

A task and schedule planning application built with **.NET** and **Angular**, designed to organize work activities, deadlines, and priorities.

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Backend | C#, ASP.NET Core, Web API |
| Frontend | Angular, TypeScript, Angular Material |
| Database | SQL Server / MySQL |

## Features

- Create and manage tasks with deadlines
- Priority levels and status tracking
- Calendar/schedule view
- Clean, responsive UI
